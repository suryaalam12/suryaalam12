Hasil Perkerjaan
  • Retifikasi Polygon PBB Berdasarkan Citra Satelit
  • - Mengkonversi bidang objek PBB dari format CAD/Mapinfo (undefined coordinate system) ke format Shapefile sebanyak 306 file menggunakan QGIS.
  • - Meluruskan dan mengkoreksi posisi bidang objek PBB ke posisi sebenarnya berdasrakan kenampakan bentuk rupa bumi berdasarkan citra satelit.
  • - Menggunakan bantuan tools di ArcGIS yaitu Spatial Adjusment dengan sangat teliti agar blok peta PBB tepat pada posisi sebenarnya.

  • Pembuatan Peta Zona ZNT dengan tampilan WebGis
  • - Menggabungkan nilai hasil poenilaian zona nilai tanah dengan data shapefile per desa menggunakan QGIS.
  • - Membuat tampilan data nilai zona tanah secara representatif yang dilengkapi dengan hover, jika user ingin mengetahui properti di dalam bidang.
  • - Meletakan data pada server agar mudah diakses secara bersamaan oleh banyak user.
